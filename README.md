# VoteWise - Interactive Election Education Platform

VoteWise is a web application designed to increase civic literacy in India. It aims to empower citizens by providing accessible, engaging, and neutral information about the electoral process.

## Features

- **Interactive Election Timeline:** Visualizes the entire election process from notification to results.
- **AI Chat Assistant:** Provides neutral, accessible civic guidance and answers questions about voting and elections.
- **Gamified Civic Quiz:** A fun way to test and improve your knowledge about the democratic system.
- **Bilingual Support:** Supports both English and Hindi for broad civic impact.
- **Responsive Design:** Built with a modern, accessible, and patriotic UI.

## Technologies Used

- **Frontend:** React, TypeScript, Vite, Tailwind CSS
- **Backend/Integrations:** Anthropic API for AI features, Spring Boot (planned/integrated backend)
- **Package Manager:** pnpm

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- pnpm package manager

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ritish404/vote-wise-1.git
   ```

2. Navigate to the project directory:
   ```bash
   cd vote-wise-1
   ```

3. Install dependencies using pnpm:
   ```bash
   pnpm install
   ```

### Running the Project

To start the development environment, run:

```bash
pnpm dev
```

For building the project:

```bash
pnpm build
```
